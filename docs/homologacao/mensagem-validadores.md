# Mensagem para validadores — Homologação controlada do Portal NITE

Olá,

O MVP Premium do Portal NITE está pronto para homologação controlada. Esta etapa serve para validar experiência, clareza, navegação, conteúdo institucional, acessibilidade percebida e adequação visual antes de qualquer decisão de publicação ampla.

URL de preview/homologação: <https://portal-nite-breno-cs-projects-d1b25706.vercel.app?_vercel_share=roXcOKz6c3u3lnAqHBhIQgadlV6sLAQ7>

## Rotas Para Avaliação

- `/`
- `/projetos`
- `/projetos/[slug]`
- `/oportunidades`
- `/atualizacoes`
- `/contato`

## Como Validar

- Acesse as rotas em desktop e mobile.
- Verifique se a navegação, Header, Footer e CTAs estão claros.
- Observe se os estados vazios e pendentes estão honestos.
- Confirme se não há conteúdo demonstrativo parecendo real.
- Registre feedback por rota sempre que possível.

## Classificação Do Feedback

Use uma das prioridades abaixo para cada apontamento:

- `bloqueante`: impede a homologação controlada ou causa erro grave.
- `ajuste antes de release`: deve ser corrigido antes de uma publicação mais ampla.
- `melhoria futura`: pode ser planejado para ciclo posterior.

## Fora Do Escopo Desta Homologação

- Formulário funcional.
- Backend, storage, notificação ou envio real.
- Timeline real.
- Posts reais.
- Fotos e depoimentos.
- Governança mínima de conteúdo.

Esta homologação valida o MVP Premium em ambiente controlado; não representa publicação ampla ou aprovação institucional final.
