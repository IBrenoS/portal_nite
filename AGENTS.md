# Instrucoes para Agents

## Leitura obrigatoria de especificacao

Antes de iniciar qualquer tarefa:

1. Abrir o arquivo `specs_svg.md`.
2. Ler o arquivo do inicio ao fim, sem pular secoes.
3. Usar o `specs_svg.md` como referencia principal durante a execucao da tarefa.

## Regra para marcos

1. Nao marcar marco como concluido automaticamente.
2. Marcar o marco somente apos confirmacao humana explicita.
3. Diferenciar conclusao de task e fechamento de milestone:
   - Milestone so pode ser marcado como concluido apos confirmacao humana explicita.
