# Definition of Ready

Uma tarefa so pode ser iniciada quando cumprir estes criterios:

- [ ] A tarefa esta vinculada a uma spec.
- [ ] O objetivo da tarefa esta claro.
- [ ] O escopo esta definido.
- [ ] Os criterios de aceitacao existem.
- [ ] Os dados ou conteudos necessarios estao disponiveis ou simulados conscientemente.
- [ ] As dependencias foram identificadas.
- [ ] A tarefa possui resultado verificavel.
- [ ] Ha clareza sobre o que esta fora do escopo.
